Hexo 備忘誌

備忘一下hexo指令
不然久久沒寫網誌都忘記要怎麼發文了...

新增文章：

hexo new "title"

預覽文章：

hexo generate
hexo server

發佈文章：

hexo deploy


另外備忘一下在Windows上安裝的過程:

  • 1. 安裝Node.js

  • 2. 安裝Git

  • 3. 打開Node.js command prompt, 輸入

    • 　　就會開始安裝hexo，跑完後可以打hexo看有沒有反應

  • 4. 把目錄cd到你要放blog的地方，輸入

  • 5. 加入套件

  • 6. 產生blog然後在本機端跑跑看

    • 　　完成後去 http://localhst:4000 看看畫面正不正常

  • 7. 搞定Github的憑證
    　　https://help.github.com/articles/generating-ssh-keys/

  • 8. 在Github上開一個名為YourID.github.io的repository

  • 9. 打開_config.yml, 設定deploy資訊

  • 10. url部分也改成https的repo (這個不確定要不要做...)

  • 11. (重要!!!) cd到你的blog目錄，輸入

  • 12. 如果失敗或是漏步，記得可以先刪掉.deploy資料夾再試試

    • 查過所有教學文，都沒人講第11步要先init(啊啊啊我的三小時鬼打牆...)
    如果你也不幸直接deploy然後爆了的話
    就先刪掉.deploy再試一次
